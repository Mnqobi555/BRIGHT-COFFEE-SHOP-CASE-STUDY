# BRIGHT-COFFEE-SHOP-CASE-STUDY

**OUR GOAL** 
Provide a newly appointed CEO with data-driven insights to improve revenue and optimize product strategy.

Key Business Questions
1. Which products generate the most revenue?
2. When does the store perform best during the day?
3. What are sales trends across products and time?
4. What actions can improve sales performance?

Project Steps

Step 1 — Plan & Architect
Designed the data flow and ETL architecture; defined core KPIs.

Step 2 — Process & Clean Data
Loaded raw Excel → CSV → Databricks; standardized pricing and created 30-min time buckets.

Step 3 — Analyze in Excel
Built pivot tables and dashboards covering revenue by category, peak sales times, and best/worst sellers.

Step 4 — Present to Executives
Delivered a data story/presentation with visuals and growth recommendations.

 Tools Used
- Data Processing: SQL and Databricks
- Visualization: Excel 
- Planning & Presentation: Miro and PowerPoint
- Project tracking: Canva

Key Recommendations
- Run marketing campaigns during low-sales time slots.
- Stock up on best-selling items.
- Use special offers to boost underperforming products.
- Introduce loyalty programs tied to peak customer times.
